# AUDITORIA DB
### Instalación 
En entornos Unix, solo tienes que correr estos comandos para ejecutar el proyecto:

```
./install.sh
```

```
./start.sh
```
En windows, quizá funcione usando wsl
### Requisitos
- python3.11
- virtualenv
- node 21
- Llenar los archivos ```client/.env``` y ```backend/.env```
